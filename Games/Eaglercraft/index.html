Eaglercraft (Minecraft)

Creator: Mojang Studios

Browser port creator: Lax1Dude

Minecraft is a game about breaking and placing blocks.
At first, people built structures to protect against nocturnal monsters,
but as the game grew players worked together to create wonderful, imaginative things.

Eaglercraft is a port of Minecraft to JavaScript and WASM created by Lax1Dude and PaytonPlayz585
